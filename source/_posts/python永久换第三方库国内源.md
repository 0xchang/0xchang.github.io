## python永久换第三方库国内源

```powershell
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

 国内靠谱的 pip 镜像源有：

(1)清华: https://pypi.tuna.tsinghua.edu.cn/simple

(2)豆瓣: http://pypi.douban.com/simple/

(3)阿里: http://mirrors.aliyun.com/pypi/simple/
